# 野球の作戦ボード

`base image` フォルダのベース画像を使った、ブラウザ用の野球作戦ボードです。

## 使い方

1. `index.html` をブラウザで開く
2. 描画モードが OFF のときに、選手・ボール・ランナーのマーカーをドラッグする
3. 描画モードを ON にして、ボード上に作戦線を書く
4. `線を消す` または `マーカーを戻す` で整理する

## ファイル

- `index.html`: 画面構成と初期マーカー配置
- `style.css`: レイアウトとマーカーの見た目
- `script.js`: ドラッグ操作と描画処理
- `base image/...png`: ベース画像
